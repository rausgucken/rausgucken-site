Schnellstart

In 2 Minuten eingebettet

Kopieren Sie diesen Code und fügen Sie ihn in Ihren Seitenquelltext ein.

     data-rausgucken-limit="5"></div>
<script src="https://www.rausgucken.de/widget.js" async></script>

Häufige Fragen

Was Sie wissen sollten

Kostet das etwas?
Nein. Das Widget ist kostenlos. Keine Anmeldung erforderlich.

Werden die Daten aktuell gehalten?
Ja. rausgucken.de aktualisiert täglich automatisch. Das Widget zeigt immer den aktuellen Stand.

Verlangsamt das Widget meine Website?
Das Script lädt async — es blockiert nie das Laden Ihrer Seite.

Darf ich das Widget verwenden?
Ja, für nichtkommerzielle Websites. Der „Powered by rausgucken.de“-Hinweis muss sichtbar bleiben.
